My Journey

My journey began in 2020 when I started studying Computer Science. During this period, I built a strong foundation in programming, logical thinking, and system design, which shaped the way I approach problem solving today.

As I progressed, I transitioned from theory into real-world coding by building practical frontend projects using JavaScript and React. This phase helped me improve my problem-solving skills and understand how to translate ideas into functional user interfaces.

Alongside my technical growth, I maintained a balanced lifestyle by staying connected with friends and engaging in social activities, which helped me remain creative, mentally refreshed, and consistent in my learning journey.

I later joined XibFnx Solutions, where I had the opportunity to work on real production-level applications. This experience exposed me to industry standards, collaboration, and the importance of writing maintainable and scalable code.

During this time, I worked extensively with React, focusing on building reusable components, scalable UI architectures, and responsive interfaces that deliver smooth and consistent user experiences across devices.

I also strengthened my understanding of API integration, state management, and performance optimization, which significantly improved my confidence and capability as a frontend developer working on real-world applications.
